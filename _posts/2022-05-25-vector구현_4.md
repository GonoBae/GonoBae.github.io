# vector의 iterator 구현해보자!! (4)

## iterator 는 무엇일까?
### 반복자
우리가 `반복`이라는 개념은 for 문을 통해 배우게 된다.

```cpp
for(int i = 0; i < vector.size(); ++i) { }
```

`반복자`라는 개념은 똑같다.
```cpp
for(iter = vector.begin(); iter != vector.end(); ++iter) { }
```

이렇게 vector 내부를 순회할 수 있게 해준다.
