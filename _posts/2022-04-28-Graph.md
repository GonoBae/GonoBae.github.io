<span style="color:yellow"> BFS 란? </span> <br>
BFS (Breadth-First-Search) 는 너비 우선 탐색 이라고 하며, Queue 를 사용한다.
**너비 우선** 이라는 말은 탐색하고자 하는 노드의 이웃 노드를 모두 방문한 다음에 이웃의 이웃을 방문하는 방식이다.
쉽게 말해, 거리에 따라 단계별로 탐색한다고 볼 수 있다.
순서는 다음과 같다.
> 너비 우선 탐색
  - Node 0
  - Node 1
  - Node 4
  - Node 5
  - Node 3
  - Node 2
